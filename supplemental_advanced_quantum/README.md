# Supplemental Advanced Quantum

This folder holds the published Advanced Quantum course PDFs together with the editable chapter source material.

- `course.pdf` is the canonical merged course PDF.
- `lecture_01.pdf` through `lecture_10.pdf` are the canonical lecture PDFs.
- `ch1/` keeps the LaTeX source material already maintained in this track.

The top-level PDFs are linked to the rebuilt course outputs so the canonical filenames in this folder stay current when the note pipeline refreshes them.
