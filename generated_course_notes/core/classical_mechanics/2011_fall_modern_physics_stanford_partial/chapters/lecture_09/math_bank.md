## Definitions And Objects
- Phase space:
  coordinates \((q_i,p_i)\), not just \(q_i\) alone.
- Poisson bracket:
  an antisymmetric, linear operation on functions of phase-space coordinates.
- Functions \(A,B,C\):
  ordinary commuting functions of the \(p\)'s and \(q\)'s.
- Canonical Poisson-bracket relations:
  the basic relations among the coordinate functions \(q_i,p_i\).
- Flow on phase space:
  a motion of points in phase space, either physical time evolution or a more general transformation.
- Hamiltonian flow:
  the special flow generated by \(H\), giving actual time evolution.
- Canonical transformation:
  a transformation of phase-space coordinates that preserves the Poisson-bracket structure.
- Infinitesimal transformation:
  \(Q=q+\delta q,\;P=p+\delta p\), with quadratic and higher powers of \(\delta q,\delta p\) neglected.
- Generator \(G(p,q)\):
  the phase-space function whose Poisson brackets generate an infinitesimal canonical transformation.
- Symmetry:
  a canonical transformation whose flow leaves the Hamiltonian unchanged.
- Kronecker delta:
  \(\delta_{ij}=1\) if \(i=j\), \(0\) otherwise.

## Derivation Steps
1. From antisymmetry to \(\{P,P\}=0\)
1. Start with antisymmetry: \(\{P,P\}=-\{P,P\}\).
2. Move the right-hand side to the left: \(2\{P,P\}=0\).
3. Conclude \(\{P,P\}=0\).
4. The same logic gives \(\{Q,Q\}=0\).

1. From the derivative definition to \(\{Q_i,P_j\}=\delta_{ij}\)
1. Use the derivative form of the Poisson bracket.
2. Differentiate \(Q_i\) with respect to \(Q_k\) and \(P_k\), and \(P_j\) with respect to \(Q_k\) and \(P_k\).
3. Only the mixed \(Q_i\)-with-\(P_j\) term survives.
4. The result is \(1\) when \(i=j\) and \(0\) otherwise.
5. Therefore \(\{Q_i,P_j\}=\delta_{ij}\).

1. Why functions of only \(Q\) bracket to zero with functions of only \(Q\)
1. In the Poisson-bracket definition, each term contains a derivative with respect to \(P\).
2. If both functions depend only on the \(Q\)'s, every \(P\)-derivative vanishes.
3. Hence \(\{F(Q),G(Q)\}=0\).
4. The same reasoning gives \(\{F(P),G(P)\}=0\).

1. Proof sketch that \(\{F(q),P\}=dF/dq\)
1. State the target identity for a function of one coordinate.
2. By linearity, reduce to monomials in \(q\).
3. Check \(F(q)=1\): use the product rule on \(\{1^2,P\}\) to show \(\{1,P\}=0\).
4. Check \(F(q)=q\): this is the defining bracket \(\{q,P\}=1\).
5. Check \(F(q)=q^2\): use the product rule to get \(\{q^2,P\}=q\{q,P\}+\{q,P\}q=2q\).
6. Induct on \(q^n\).
7. Extend from polynomials to smooth functions by polynomial approximation, as the lecture suggests.

1. Hamiltonian evolution from the Poisson algebra for \(H=P^2/2M\)
1. Use \(\dot A=\{A,H\}\).
2. Set \(A=P\): since \(H\) is a function only of \(P\), \(\dot P=\{P,H\}=0\).
3. Set \(A=Q\): use the product rule on \(\{Q,P^2\}\).
4. Compute \(\{Q,P^2\}=P\{Q,P\}+\{Q,P\}P=2P\).
5. Divide by \(2M\) to get \(\dot Q=P/M\).

1. Canonical-transformation test for the scaling examples
1. Try \(P=2p,\;Q=2q\).
2. Compute \(\{P,Q\}=4\{p,q\}=4\), so the canonical relation is not preserved.
3. Try \(P=\tfrac12 p,\;Q=2q\).
4. Compute \(\{P,Q\}=\tfrac12\cdot 2\,\{p,q\}=1\).
5. Conclude the second map preserves the one-pair Poisson structure.

1. Canonical-transformation test for the phase-space rotation
1. Write \(P=\cos\theta\,p+\sin\theta\,q\) and \(Q=-\sin\theta\,p+\cos\theta\,q\).
2. Use antisymmetry to know \(\{P,P\}=0\) and \(\{Q,Q\}=0\).
3. Expand \(\{P,Q\}\) by linearity.
4. Terms involving \(\{p,p\}\) and \(\{q,q\}\) vanish.
5. Mixed terms produce \(\cos^2\theta\,\{p,q\}-\sin^2\theta\,\{q,p\}\).
6. Since \(\{q,p\}=1\) and \(\{p,q\}=-1\) or vice versa depending on order convention, the two contributions combine to give the canonical value.
7. Conclude the rotation preserves the Poisson structure.

1. First-order canonical condition for infinitesimal transformations
1. Write \(Q=q+\delta q\), \(P=p+\delta p\).
2. Expand \(\{Q,P\}\) using linearity.
3. Drop \(\{\delta q,\delta p\}\) because it is quadratic in small quantities.
4. Demand \(\{Q,P\}=\{q,p\}\).
5. Obtain \(\{\delta q,p\}+\{q,\delta p\}=0\).
6. Therefore \(\{\delta q,p\}=-\{q,\delta p\}\).

1. Generator construction for infinitesimal canonical transformations
1. Propose \(\delta q=\epsilon\{q,G\}\) and \(\delta p=\epsilon\{p,G\}\).
2. Rewrite these using the one-pair differentiation rules:
   \(\delta q=\epsilon\,\partial G/\partial p\), \(\delta p=-\epsilon\,\partial G/\partial q\).
3. Insert them into the first-order canonical condition.
4. The left-hand side becomes equality of mixed second derivatives of \(G\).
5. Therefore the canonical condition is satisfied.

1. Change of any observable under a \(G\)-generated flow
1. For any function \(A(p,q)\), write \(\delta A=(\partial A/\partial p)\delta p+(\partial A/\partial q)\delta q\).
2. Substitute \(\delta p=-\epsilon\,\partial G/\partial q\) and \(\delta q=\epsilon\,\partial G/\partial p\).
3. Recognize the resulting expression as \(\epsilon\{A,G\}\).
4. Conclude that Poisson bracketing with \(G\) generates the flow.

1. Symmetry implies conservation
1. Define symmetry by requiring that the \(G\)-flow not change the Hamiltonian.
2. Therefore \(\delta H=\epsilon\{H,G\}=0\), hence \(\{H,G\}=0\).
3. Use antisymmetry to rewrite this as \(\{G,H\}=0\).
4. Apply \(\dot G=\{G,H\}\).
5. Conclude \(\dot G=0\): the generator is conserved.

1. Free-particle angular-momentum example
1. Take \(H=(p_x^2+p_y^2)/(2m)\) and \(G=xp_y-yp_x\).
2. Expand \(\{G,H\}\) into terms involving \(\{xp_y,p_x^2\}\), \(\{xp_y,p_y^2\}\), \(\{yp_x,p_x^2\}\), and \(\{yp_x,p_y^2\}\).
3. Terms like \(\{x,p_y\}\) and \(\{y,p_x\}\) vanish.
4. The nonzero contributions reduce to \(p_y p_x\) and \(-p_y p_x\).
5. These cancel, giving \(\{G,H\}=0\).
6. Therefore angular momentum is conserved for the free particle.

1. What it means to use \(G\) itself as a Hamiltonian
1. Set \(H=xp_y-yp_x\).
2. Compute \(\dot x=\{x,H\}=-y\).
3. Compute \(\dot y=\{y,H\}=x\).
4. Recognize the motion as circular motion.
5. Read the earlier vanishing bracket reciprocally: if this \(G\) is the Hamiltonian, then the old free-particle energy is conserved.

## Notation Choices
- Use \(\{A,B\}\) consistently for Poisson brackets.
- Use lowercase \(q_i,p_i\) for the standard canonical coordinates of the system.
- Use uppercase \(Q_i,P_i\) only for transformed coordinates, especially in the canonical-transformation examples.
- When reproducing the board-supported one-pair discussion, it is acceptable to write \(q,p\) and later normalize to indexed \(q_i,p_i\) when the notes broaden to many degrees of freedom.
- Use \(\delta_{ij}\) for the Kronecker delta.
- Use dot notation for Hamiltonian time evolution:
  \(\dot A=\{A,H\}\).
- For infinitesimal transformations, use
  \(Q=q+\delta q\), \(P=p+\delta p\),
  and a separate small scalar \(\epsilon\) in
  \(\delta q=\epsilon\{q,G\}\), \(\delta p=\epsilon\{p,G\}\).
- Keep the order conventions exactly as in the lecture where they matter:
  \(\dot A=\{A,H\}\),
  \(\delta A=\epsilon\{A,G\}\),
  and the warning that the order inside the bracket matters.
- Use ordinary multiplication for functions, and treat it as commutative in the classical notes.
- In the one-variable differentiation identity, use \(\{F(q),P\}=dF/dq\) when staying close to the frame; in the many-variable normalized text, prefer partial derivatives.
- Avoid calling the functions “scalars” in the main mathematical narration; “functions” or “commuting numerical-valued functions on phase space” matches the lecture more safely.
- Reserve “generator” for the function \(G(p,q)\) whose Poisson bracket produces a flow.
- Define “canonical transformation” by preservation of the canonical Poisson-bracket relations, not by an external textbook criterion.
